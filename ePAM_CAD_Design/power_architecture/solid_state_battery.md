# Solid-State Battery Subsystem

## Technology

| Parameter | Specification |
|-----------|--------------|
| **Chemistry** | Lithium solid-state (ceramic electrolyte) |
| **Energy density** | 500 Wh/kg (vs 250 Wh/kg Li-ion) |
| **Cycle life** | >2,000 cycles (80% capacity retention) |
| **Fast charge** | 10-80% in 12 minutes (4C rate) |
| **Safety** | Non-flammable, no thermal runaway |
| **Operating temp** | -30C to +60C |
